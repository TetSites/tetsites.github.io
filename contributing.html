How to Contribute

  • New Folder Setup: To add a new website folder (your subdomain path), create a pull request (PR) with your folder and site files.
  • Allowlist Control: When adding a new folder, an allowlist file will be created to track who can edit it.
  • Editing Existing Folders: Only users listed in the allowlist for a folder or global admins may edit that folder.
  • Review Process: All PRs will be manually reviewed for permission and quality before merging.

Code Style & Formatting

Please keep your HTML clean, readable, and consistent with the existing site style (dark theme, simple layout).

Need Help?

If you have questions or want to request access, contact a project admin listed in allowlist/global.md.

We appreciate your support and look forward to your contributions!
